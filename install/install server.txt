If you want the RMI server to start when Tomcat does:
--------------------------------------------------------
WARNING! there is a bug in either JavaRMI or tomcat itself, but in order to make RMI work
under Tomcat make shure that Tomcat is not installed under a directory-tree with spaces in the dir-names!
Copy lib/masserr.jar to [tomcatdir]/server/lib

Edit [tomcatdir]/conf/server.xml

add the line
<Listener className="se.tdt.bobby.wodcc.server.catalina.LifecycleListener"/>
inside the <server> tag.
